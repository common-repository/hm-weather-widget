== Description ==
 Hm weather widget is a simple  and lightweight plugin that will help you to show weather condition on your site . You can Display individual city weather or your own city with geolocation. 
 option for Display All day Condition and forecast
 Also option for Full forecast through api.

 All data Display Through Wunderground Api . Wunderground is weather forecast site. 
 GeoPlugin use for Detect own  position , city . 
 GeoPlugin only work on live server not in localhost. 





== Installation ==

1. Add plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to widget secction and add it any sidebar
1. Fill the state and city (Required)
1. check other option if you want to show other information
